Firebase 연결 설정

이 앱은 Firebase(구글의 서버 서비스)를 저장소로 써요. 아래 순서대로 한 번만 설정하면 돼요.
1) firebase.google.com/console 접속 → 프로젝트 만들기 2) 왼쪽 메뉴에서 "Firestore Database" 만들기 (프로덕션 모드로 시작해도 OK, 아래 규칙을 붙여넣을 거예요) 3) 왼쪽 메뉴 "Authentication" → Sign-in method → 이메일/비밀번호 사용 설정 4) Authentication → Users 탭에서 팀원 이메일/임시비밀번호를 직접 추가 (회원가입 화면은 따로 없어요, 관리자가 직접 등록) 5) 프로젝트 설정(톱니바퀴) → 일반 → "내 앱" → 웹 앱 추가 → 나오는 firebaseConfig 객체를 통째로 복사해서 아래에 붙여넣기
Firestore 규칙(꼭 설정)! Firestore Database → 규칙 탭에 아래 내용을 붙여넣어 로그인한 사람만 읽고 쓸 수 있게 해주세요.
rules_version = '2'; service cloud.firestore { match /databases/{database}/documents { match /{document=**} { allow read, write: if request.auth != null; } } }

총 재고

상품별 · 창고별 재고. 모든 창고를 더한 값이 총 재고예요.

창고 관리

공통창고, 대리점, 통합매장 등 재고가 쌓이는 장소들을 등록해요.

매입 등록 (+)

특정 창고에 재고를 더해요. 신규 입고, 초기 재고 등록에 써요.

일괄 입력

엑셀에서 [상품명] [탭] [창고명] [탭] [수량] 세 칸을 복사해서 그대로 붙여넣으면 한 번에 등록돼요.

매출 처리 (-)

특정 창고에서 재고를 차감해요.

창고 이동

공통창고 → 대리점처럼, 한 창고에서 다른 창고로 옮겨요. 출발 창고에서 빠지고 도착 창고에 동시에 더해져요 (총 재고는 그대로예요).
